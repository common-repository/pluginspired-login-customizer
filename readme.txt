== Description ==

= What PlugInspired Login Customizer Can Do =

PlugInspired Login Customizer allows you to easily customize the default WordPress login, registration and forgot password pages for your site or blog. Including the colors, logo, and background.

The Colors & Styles options allow you to change the colors of the login box, field labels, buttons, links and more.

The Logo options include a field to upload a new image or select an image from the WordPress media library, a field that allows you to change the link title (seen on mouse-over), and the logo link.

The Background options include the ability to change the background color, add a background image, set the background image size, and control whether or not the background repeats (great for smaller, tiled backgrounds).

== Installation ==

= Installing PlugInspired Login Customizer =

= From your Dashboard =

1. Go to Plugins > Add New.
2. Search for `PlugInspired Login Customizer`.
3. Locate `PlugInspired Login Customizer` by `PlugInspired` in the list.
4. Click Install and wait for the installation to finish.
5. If the installation is successful, click the `activate plugin` link.

= From FTP Upload =

1. Download PlugInspired Login Customizer from the [WordPress Plugin Repository](https://downloads.wordpress.org/plugin/pluginspired-login-customizer.zip)
2. Extract the ZIP archive somewhere on your computer that you can find easily
3. Upload the extracted `pluginspired-login-customizer` directory to your site's `/wp-content/plugins/` directory
4. Activate the plugin on the 'Plugins' page in your WordPress Dashboard

= Using PlugInspired Login Customizer =

After installing PlugInspired Login Customizer, you can access the plugin's settings two ways.

1. By clicking the `Settings` link on the Plugins page in your Dashboard
2. By clicking the `Login` link on the menu in your Dashboard
